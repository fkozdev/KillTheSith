
# KillTheSith

Python Kullanılarak yapılmış basit bir oyun.
## Kullanılan Teknolojiler

**Dil:** Python

**Modüller:** pygame

  
## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/fuekihigh/KillTheSith
```

Proje dizinine gidin

```bash
  cd KillTheSith
```

Kodu çalıştırın

```bash
  python3 Game.py
```
